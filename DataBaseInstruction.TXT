Требования к БД:

В целом:
url = jdbc:postgresql://localhost:5432/
database = holiday
username = postgres
password = password
type = postgresql

Краткое описание:

1. Скачиваем и устаналиваем приложение базы данных с официального сайта - https://www.postgresql.org/download/.
На сайте так же представлены инструкции и скрипты для различных типов операциооных систем, вплоть до ручной установкви на ОС Windows.
    1.1. Принимаем стандартный порт базы данных - 5432
    1.2. Задаём пользователю postgres пароль password

2. После установки базы необходимо создать базу данных с именем "holiday". К примеру через стандартное приложения с GUI для Windows - PgAdmin.
